# Valentine's Day Website for Krithu

A cute, interactive single-page site you can host on GitHub Pages.

## Features

- Password gate question:
  - Prompt: `what is the bestest day in the whole world?!`
  - Correct answer: `06/14/2024`
- Main question: `Will you be my valentine?`
- `No` button runs away when hovered.
- `Yes` button grows bigger each time `No` tries to escape.
- Animated Border Collie-style dog (`Chitti`) with floating hearts.
- Works on desktop and mobile.

## Run locally

Open `/Users/yashgarala/Development/Vday Project/surprise/index.html` in your browser.

## Host on GitHub Pages

1. Push this repo to GitHub.
2. In your GitHub repo, open **Settings > Pages**.
3. Under **Build and deployment**, set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or your default branch)
   - **Folder**: `/ (root)`
4. Click **Save**.
5. Wait about 1-3 minutes, then open your Pages URL.
